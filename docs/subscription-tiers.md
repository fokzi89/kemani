## Plan Details

### Free Plan (₦0/month)

**Ideal for:** Small businesses testing the platform, individual vendors, kiosks

**Limits:**
- 1 branch only
- 3 staff users maximum
- 100 products maximum
- 500 transactions per month
- 0% commission (no e-commerce available)

**Features:**
- Basic POS functionality
- Customer management
- Sales reporting (basic)
- Receipt generation (PDF/print)
- Email support

**Restrictions:**
- No e-commerce storefront
- No AI chat
- No delivery management
- No branch transfers
- No analytics dashboard

## Feature Flags (JSONB)

Each subscription can have additional features enabled via the `features` JSONB field:

```json
{
  // E-Commerce
  "ecommerce_chat": true,
  "ai_chat": true,

  // Analytics
  "advanced_analytics": true,
  "custom_reports": true,
  "data_export": true,

  // Integrations
  "api_access": true,
  "woocommerce_sync": true,
  "shopify_sync": true,
  "whatsapp_business_api": true,

  // Currency & Payments
  "multi_currency": true,
  "custom_payment_gateway": true,

  // Infrastructure
  "custom_integrations": true,
  "dedicated_infrastructure": false,
  "on_premise": false
}
```

## Upgrading Plans

### From Free to Basic

```sql
-- Upgrade tenant to Basic plan
UPDATE tenants
SET subscription_id = (
    SELECT id FROM subscriptions
    WHERE plan_tier = 'basic'
    AND tenant_id = 'tenant-uuid'
)
WHERE id = 'tenant-uuid';
```

### From Basic/Pro to Enterprise

```sql
-- Upgrade to Enterprise
UPDATE subscriptions
SET plan_tier = 'enterprise',
    max_branches = 999999,
    max_staff_users = 999999,
    max_products = 999999,
    monthly_transaction_quota = 999999,
    commission_rate = 1.00,
    monthly_fee = 50000.00,
    features = features || '{"advanced_analytics": true, "multi_currency": true}'::jsonb
WHERE tenant_id = 'tenant-uuid';
```

### To Enterprise Custom

Requires manual sales process:

1. Customer contacts sales
2. Needs assessment & pricing
3. Contract signing
4. Plan creation with custom parameters
5. Custom domain setup
6. Theme customization
7. Migration & onboarding

## Check Current Plan

```sql
-- Get tenant's current plan and features
SELECT
    t.name as tenant_name,
    t.slug,
    t.ecommerce_enabled,
    t.custom_domain,
    s.plan_tier,
    s.monthly_fee,
    s.commission_rate,
    s.max_branches,
    s.max_products,
    s.features
FROM tenants t
JOIN subscriptions s ON t.subscription_id = s.id
WHERE t.id = 'tenant-uuid';
```

## Billing

- **Billing Cycle:** Monthly (from subscription start date)
- **Payment Methods:** Bank transfer, card, mobile money
- **Auto-renewal:** Yes (with 7-day grace period)
- **Prorating:** Yes (when upgrading mid-cycle)
- **Downgrading:** Effective next billing cycle
- **Cancellation:** 30-day notice required

## Commission Structure

Commissions apply to e-commerce orders only:

| Plan | Commission Rate | Maximum Per Order | Applied To |
|------|----------------|-------------------|------------|
| Free | 0% | N/A | No e-commerce available |
| Basic | 0% | N/A | No e-commerce available |
| Pro | 1.5% | ₦500 cap | E-commerce orders |
| Enterprise | 1% | ₦500 cap | E-commerce orders |
| Enterprise Custom | 0.5% (Negotiable) | ₦500 cap | E-commerce orders |

### Commission Cap Examples

All plans have a **₦500 maximum commission per order**, regardless of order value:

**Pro Plan (1.5% rate, ₦500 cap):**
- ₦10,000 order → ₦150 commission (1.5%)
- ₦33,333 order → ₦500 commission (1.5% = ₦500, at cap)
- ₦50,000 order → ₦500 commission (1.5% would be ₦750, capped at ₦500)
- ₦100,000 order → ₦500 commission (capped)

**Enterprise Plan (1% rate, ₦500 cap):**
- ₦20,000 order → ₦200 commission (1%)
- ₦50,000 order → ₦500 commission (1% = ₦500, at cap)
- ₦100,000 order → ₦500 commission (capped)
- ₦500,000 order → ₦500 commission (capped)

**Enterprise Custom Plan (0.5% rate, ₦500 cap):**
- ₦50,000 order → ₦250 commission (0.5%)
- ₦100,000 order → ₦500 commission (0.5% = ₦500, at cap)
- ₦1,000,000 order → ₦500 commission (capped)

**Note:** In-store POS transactions have no commission fees.
